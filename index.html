Manual de Comandos Linux

FIND

Busca archivos y directorios según distintos criterios.

Opciones

  • -name
  • -perm
  • -mtime
  • -exec
  • -size

Ejemplos

find . -name "*.txt"
find /home -perm 644
find /var/log -mtime -7
find . -name "*.log" -exec rm {} \;
find . -size +10M

PS

Muestra procesos en ejecución.

Opciones

  • ps
  • ps aux
  • ps -u usuario

Ejemplos

ps
ps aux
ps -u root

GREP

Busca texto dentro de archivos.

Opciones

  • -i
  • -v
  • -r

Ejemplos

grep -i "error" archivo.log
grep -v "INFO" archivo.log
grep -r "main" src/
